# HD Screen Share Unlocker

A BetterDiscord plugin that enables HD screen sharing without requiring a Nitro subscription.

## Features

- Unlock 1080p/4K screen sharing
- No Nitro subscription required
- Easy to enable/disable

## Installation

1. Download `HDScreenShare.plugin.js`
2. Place it in your BetterDiscord plugins folder:
   - **Windows**: `%AppData%/BetterDiscord/plugins/`
   - **macOS**: `~/Library/Application Support/BetterDiscord/plugins/`
   - **Linux**: `~/.config/BetterDiscord/plugins/`
3. Enable the plugin in BetterDiscord settings

## How It Works

The plugin temporarily sets your Discord premium type to Nitro (type 2) when enabled, unlocking HD streaming features. When disabled, it restores your original premium type.
